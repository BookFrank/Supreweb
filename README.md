## 一. 简介 ##

Supreweb，一款自己编写的Java Web框架，参考<<架构探险>>。
通过编写这个框架，加强自己对于Java Web 框架和 Spring 的工作生命周期的理解。
同时通过自己的不断学习，不断完善框架特性，不断改进。


## 二. 特性 ##

#### 2.1 模块介绍
| 模块名                 | 介绍                    |
| ------------------- | --------------------- |
| supre-demo  | 引入 framework，演示框架的使用          |
| supre-framework | SupreWeb 框架核心代码 |

#### 2.1 模块介绍（不断完善中...）

- 基于 Alibaba FastJson
- 支持注解 @Controller @Service等
